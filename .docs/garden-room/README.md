## Summary

### Current State
- 2 sections: Hero + Gallery
- No product information, no FAQs, no testimonials, no trust signals
- Weak SEO footprint (minimal text content)

### Proposed State
- 8 sections: Hero → Product Range → Why Steel → Planning Guide → Gallery → Testimonials → FAQ → CTA
- Showcases new 15/25/35/45m² product line
- 45m² "Coming Soon" with Register Interest captures legislation-aware leads
- FAQPage + Product structured data (no competitor has either)
- Only 1 new component needed (`ProductRangeGrid`); all others reuse existing UI library

### Key Differentiators vs Competitors
1. Size-based product categorization (unique in Irish market)
2. Planning permission dedicated section (highest search volume keyword)
3. 45m² legislation awareness (no competitor addresses this)
4. FAQPage schema (untapped rich snippet opportunity)
5. Multiple conversion CTAs throughout page (not just hero)
