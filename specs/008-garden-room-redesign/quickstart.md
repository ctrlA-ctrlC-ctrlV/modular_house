# Quickstart: Garden Room Page Redesign

**Feature**: 008-garden-room-redesign
**Branch**: `008-garden-room-redesign`

## Prerequisites

- Node.js 18+ and pnpm installed
- Repository cloned and on the `008-garden-room-redesign` branch

## Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm --filter @modular-house/web dev
```

The garden room page is available at `http://localhost:5173/garden-room`.

## Implementation Order

### Step 1a: Create ProductRangeGrid Component

**Location**: `packages/ui/src/components/ProductRangeGrid/`

Create the new component with:
- `ProductRangeGrid.tsx` — Component implementation following the contract in `contracts/product-range-grid.md`
- `ProductRangeGrid.css` — BEM-styled CSS with responsive grid (4 → 2 → 1 columns)
- `ProductRangeGrid.test.tsx` — Unit tests for rendering, badge display, and CTA link generation

Export from `packages/ui/src/index.ts`.

### Step 1b: Create AccordionFAQ Component

**Location**: `packages/ui/src/components/AccordionFAQ/`

Create the new component with:
- `AccordionFAQ.tsx` — Expandable FAQ component following the contract in `contracts/accordion-faq.md`
- `AccordionFAQ.css` — BEM-styled CSS with max-height transitions
- `AccordionFAQ.test.tsx` — Unit tests for expand/collapse, ARIA attributes, keyboard interaction

Export from `packages/ui/src/index.ts`.

### Step 2: Rewrite GardenRoom.tsx

**Location**: `apps/web/src/routes/GardenRoom.tsx`

Replace the current 2-section layout with 8 sections in order:
1. Hero (HeroBoldBottomText) + anchor link overlay
2. Product Range (ProductRangeGrid) — white bg
3. Why Steel Frame (FeatureSection) — beige bg
4. Planning Permission (TwoColumnSplitLayout) — white bg
5. Gallery (FullMassonryGallery) — beige bg
6. Testimonials (TestimonialGrid) — white bg
7. FAQ (AccordionFAQ) — beige bg
8. CTA (ContactFormWithImageBg) — dark overlay

Reference Landing.tsx for section composition patterns and renderLink usage.

### Step 3: Update SEO Metadata

**Location**: `apps/web/src/routes-metadata.ts`

Update the `/garden-room` route entry:
- Title: optimized for SEO (under 70 chars)
- Description: under 160 chars, mentioning sizes and planning permission
- Add FAQPage schema with all 6 Q&A pairs
- Update Product schema with 4 size-based offers
- BreadcrumbList and WebPage already generated by `generatePageSchema()`

### Step 4: Extend Contact Page Query Params

**Location**: `apps/web/src/routes/Contact.tsx`

Add `useSearchParams()` to read `?product=` and `?interest=` params. Map the product param to pre-select the `preferredProduct` dropdown.

### Step 5: Optimize Images

```bash
# Generate WebP/AVIF variants for any new images
pnpm --filter @modular-house/web optimise:images
```
