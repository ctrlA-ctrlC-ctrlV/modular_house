## Design Principles

1. **Warm Premium** — Copper-brown primary and warm beige backgrounds convey natural materials and craftsmanship, while dark steel tones communicate structural strength.
2. **Editorial Clarity** — Large hero imagery with Special Gothic Condensed One headings creates a magazine-like editorial feel. Body copy in Inter ensures readability.
3. **Mobile-First Responsive** — Every component degrades gracefully from 4-column desktop to single-column mobile with carousel alternatives.
4. **Accessible by Default** — Focus rings, reduced-motion support, semantic HTML, ARIA labels, and keyboard navigation are built into every component.
5. **Performance-Optimized** — Images served in AVIF/WebP with fallbacks. Fluid typography avoids layout shifts. Async font loading prevents FOIT.
